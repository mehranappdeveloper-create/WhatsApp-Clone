# Flutter WhatsApp Clone UI

A Built entirely with **Flutter**. This project focuses solely on designing a sleek and modern chat interface, mimicking the look and feel of the popular WhatsApp app.

![images alt](https://github.com/mehranappdeveloper-create/Flutter-WhatsApp-Clone/blob/main/IMG-20251106-WA0007.jpg)

## Features
- Clean and modern UI design
- Chat screen layout
- Calls screen UI
- Status and profile screens (UI only)
- Responsive and smooth interface

## Tech Stack
- Flutter
- Dart

## Note
This project **only includes the UI**; it does not include backend functionality or real chat/call capabilities. It's designed for learning Flutter UI and prototyping purposes.
